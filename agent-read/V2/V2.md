二、V2 的核心方法定义
==================================================

请按照以下理解实现 V2：

1. 使用 LingBot 现有 future video latent generation 链路生成 future latent：
   z_hat = G_theta(h, eps_z)

2. 将 future latent z_hat 作为 condition 输入给一个新的 noise policy：
   w_a = pi_omega(h, z_hat)

3. 将 w_a 替换原本 action flow / inverse dynamics / action decoder 中默认采样的 action noise：
   a = H_psi(h, z_hat, w_a)

4. 冻结 LingBot backbone：
   - future video latent generation 部分冻结
   - inverse dynamics / action flow / action decoder 部分冻结
   - LingBot backbone 不参与训练更新

5. 只训练：
   - noise policy
   - 以及实现该实验所必须的新模块（例如 critic / value，如果你的实现采用 DSRL 风格训练）

6. 该实验本质上是：
   - 不重构 LingBot
   - 不修改 LingBot 核心模型结构
   - 只把 LingBot 作为冻结的 latent proposer + frozen action decoder
   - 在中间新增一个可学习的 noise controller
